args.py file has all arguments that can be changed if you want to use with the main.ipynb

If you want to run by saving main.ipynb as main.py:
1: Download main.ipynb as main.py
2: run python main.py with the args as required else default values have been set up

model.py has the three models:
1: CNN-Static
2: CNN-Non-Static
3: CNN Multi-Channel

preprocess.py has all the pre-processing steps which could be either used or not used

pred.py has the functions for predicting the output and displaying plots and reports
